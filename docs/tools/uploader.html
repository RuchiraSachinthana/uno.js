Uploader for Intel HEX Files

Intel hexadecimal object file format, Intel hex format or Intellec Hex is a file that commonly used for programming microcontrollers, EPROMs, and other types of programmable logic devices and hardware emulators.

In Arduino IDE, when we upload a sketch that we programmed to a microcontroller, the hex file of that sketch will be generated in temp files in your computer. That file is the only thing finally needed to upload as the program to microcontroller.

uno.js library has an Uploader() that you can upload this kind of hexadecimal files into your microcontroller directly from the web browser. This method relies on AvrgirlArduino library, that has already included within uno.js library.

To use this method create a new Uploader() and use upload() method or uploadFile() method to start uploading. For the upload() method, you have to provide the Arduino board model and hex content. And for the uploadFile() method, you can provide hex file URL as the second input and Uploader will read that file using fetch to get the hex content. 

// create uploader
const uploader = new UNO.Uploader()

// start with user gesture
document.addEventListener('click', () => {
    // upload file into Arduino Mega board
    uploader.uploadFile('mega', 'path/to/file.hex').then(() => {
        // uploading complete
    }).catch(() => {
        // upload error
    })
})

As microcontroller board model, you can provide one of following code. 

uno      - Arduino Uno
mega     - Arduino Mega
leonardo - Arduino Leonardo
micro    - Arduino Micro
nano     - Arduino Nano

Uploader() can be combined with UploadButton() in utils to make a small upload gadget easily.
