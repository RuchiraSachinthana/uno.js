Built-In Function

Most common function that you can see in the Arduino programming are also included into uno.js library. pinMode(), digitalWrite(), analogWrite(), digitalRead(), analogRead(), delay() are few of them. Let's look at those built-in functions one by one.

Digital I/O

Hanlding and reading the digital states of the controller is the main purpose of these functions. Use the pinMode() function to set the mode of a pin and provide pin number and the mode. There are three constants in the uno.js that you can use such as INPUT, OUTPUT and INPUT_PULLUP. 

await myBoard.pinMode(4, INPUT)

If you need to set multiple pin modes at the same time, it is recommanded to use object input to do that for better performance. 

await myBoard.pinMode({
  INPUT : [3, 4],
  OUTPUT : [10]
})

You can set the voltage of a pin using digitalWrite() function. Just provide the pin number and the voltage level such as HIGH and LOW. 

await myBoard.digitalWrite(6, HIGH)

Use the object input mode to handle digitalWrite() for multiple pins. 

await myBoard.digitalWrite({
  HIGH : [6, 7],
  LOW : [2, 8, 12]
})

You can use digitalRead() function to read the current voltage level of a pin. Here do not use await operator because pin vlotage states automatically updates in the front-end after any other function that comes with uno.js. So it's acctually returnning the voltage value from a memory in web browser. 

const volt = myBoard.digitalRead(10)

Analog I/O

Use analogWrite() function to set an specific voltage level to a pin. The value that you provide should be an integer between 0 and 255. And be aware about the pins that you can vary the voltage level. 

await myBoard.analogWrite(6, 200)

You can use analogRead() function to read the current voltage level of a pin. The output value will be an integer between 0 and 1023. 

const volt = myBoard.analogRead(3)

Time

To deal with the time and stop the process for a movement, you have delay() and delayMicroseconds() functions within uno.js library. Just provide an integer value while using them. 

await myBoard.delay(100)
await myBoard.delayMicroseconds(10)

Also you can request the driven time of the controller using millis() and micros() functions. 

const timeMil = await myBoard.millis()
const timeMrc = await myBoard.micros()
