Introduction: JavaScript Library for Arduino

Writing Arduino programs using JavaScript and run through your web browser is the main concept of uno.js. This library communicate with the microcontroller that has connected to your PC and reads values from sensors and modules to do any kind of calculation in web environment. Look at the following advantages you will experiance using uno.js Library.

  • Combine both JavaScript and Arduino technologies together
  • No need of installing programs to microcontroller again and again
  • Run or test programs, locally or hosted in anywhere in the internet
  • Develop larger programs without worring about the storage

To sum up, you will be able to write Arduino programs using JavaScript and test them in your web browser that connected to your controller without installing.

Microcontrollers and Web Technology

We all know that devices connects to PC (USB), transfer infromation using serial communication. Even Arduino do the same thing. We read the values that calculated from Arduino controllers through the Serial Monitor that uses the serial communication.

The idea of uno.js is generated due to the features that modern web browsers have. Today web browsers are able to access serial ports to read and write data to them.

Because of that, we can access the values that sent from an Arduino controller through your web browser. uno.js library is developed to use serial communication technology to provide commands to Arduino controllers and retrieve information such as voltage reading and other sensor readings. To do this kind of operation, an Arduino client program comes with uno.js to install to your microcontroller only for the very first time. The you can write JavaScript programs and client is there in your controller to accept your commands and do what you say.

Sensors and Modules

There are thausands of devices that you can connect to an Arduino microcontroller. Many devices provide various of redings from physical environment around us. That's make the Arduino very special to come up with new ideas. uno.js is developed to break the barrior between JavaScript platform and Arduino to experiance new level of web technology.

You will be able to controll the web environment using Arduino at the same time you controll the Arduino from the web browser. Projects in the internet will be easier to test due to no requirement of downloading and installing.

Few sensors and modules that require additional libraries are already included in the client program and there will be more soon.
