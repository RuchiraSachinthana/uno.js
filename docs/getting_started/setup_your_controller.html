Arduino Client Program of uno.js

The connection between web browser and the microcontroller happens due to the uno.js client application. It's a small Arduino program that has all the methods to read and execute commands from the serial port. uno.js library provides you a feature that you can upload this client program even through web platform. Please connect your controller to your PC and select the board model then press the upload client button. After selecting the USB device, installation will start and will be completed within few seconds. To use this feature for our developments, See the uno.js Uploader() section.

Once you have succesfully installed the client application, there is no requirement of installing anything to your Arduino microcontroller otherwise if you change your uno.js version.

Manual Installation

What you upload here is a compiled version of your sketch that known as Intel HEX files. Usually Arduino IDE generates this kind of HEX file when you press upload and flashes that file into your controller. Use this link to view all the HEX files for uno.js client. To install this file mannually, you have to use a small tool like XLoader.

Install Using Arduino IDE

If you are interested in modifying the client application before uploading to your microcontroller, you can download the sketch file and the required libraries from here. Download all the files and remember to include the required libraries in your Arduino program files. 

C:\Program Files (x86)\Arduino\libraries

Be careful when you modify the client program and make sure it will associate with your uno.js front-end script even after the modifications.

Modification Instructions

If you need to remove some of features from the client sketch file for clear up some space in your microcontroller, you can follow the uno.js category codes and method codes. You can remove all the methods that related to the method or the category from the sketch.
