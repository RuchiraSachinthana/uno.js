Arduino Client Program of uno.js

The connection between web browser and the microcontroller happens due to the uno.js client application. It's a small Arduino program that has all the methods to read and execute commands from the serial port. Use this link to download the latest version of uno.js client. You will get the compiled Arduino sketch (HEX file) and you have to install it using a small tool like XLoader. Otherwise you can download sketch (INO file) and compile and upload using Arduino IDE.

Once you have succesfully installed the client application, there is no requirement of installing anything to your Arduino microcontroller otherwise if you change your uno.js version.

Installng the Client Using Arduino IDE

If you are interested in modifying the client application before uploading to your microcontroller, You can download the sketch file and the required libraries from here. Extract the ZIP file and remember to include all the required libraries in your Arduino program files. 

C:\Program Files (x86)\Arduino\libraries

Be careful when you modify the client program and make sure it will associate with your uno.js front-end script even after the modifications.

Modification Instructions

If you need to remove some of features from the client sketch file for clear up some space in your microcontroller, you can follow the uno.js category codes and method codes. You can remove all the methods that related to the method or the category from the sketch. 

CAT_CMMN 201 // built-in methods

    UJS_CLNT 0 // UNO.js client version
    PIN_MODE 1 // pinMode()
    PIN_STAT 2 // digitalRead(), analogRead()
    DT_WRITE 3 // digitalWrite()
    AL_WRITE 4 // analogWrite()
    DLY_MLSC 5 // delay()
    DLY_MRSC 6 // delayMicroseconds()
    TIM_MLSC 7 // millis()
    TIM_MRSC 8 // micros()

CAT_SNSR 202 // sensors

    USS_READ 0  // ultrasonic sensor read()
    BPS_BEGN 10 // bps init()
    BPS_READ 11 // bps read()

CAT_MODS 203 // modules

    SVR_ATCH 0  // servo.attach()
    SVR_WRTE 1  // servo.write()
    ACL_POWR 10 // adxl.powerOn()
    ACL_READ 11 // adxl.readAccel()
    LCD_BEGN 20 // lcd.begin()
    LCD_CRSR 21 // lcd.setCursor()
    LCD_PRNT 22 // lcd.print()
    LCD_CLER 23 // lcd.clear()
