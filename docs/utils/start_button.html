Start Button

To start the connection between microcontroller and the web browser, you have to initiate it using an user gesture such as mouse click. Also you have to check the browser compatibility for Web Srial API. To handle such thing, uno.js provdes you a start button that contains all those features and functionalities.

Once you create the controller using UNO.Controller() with init and stop fuctions, you can define the StartButton() and provide those things as parameters. Then the button will take care of everything to start and stop the controller and also check the browser compatibility. 

// create controller
const myBoard = new UNO.Uploader()

// init function
const init = async function() {
    // start code here
}

// stop function
const stop = async function() {
    // stop code here
}

// create start button
const button = UNO.StartButton(myBoard, init, stop)

// append button element into body
document.body.appendChild(button.element)

Providing stop function is optional. If there is no any stop function start button will execute stop() method from Controller() when you need to stop from running state.

When browser does not have the Web Srial API, this button will become read and shows as an error. And user will get to see the popup and a link to MDN docs about browser compatibility of required API.
