Communicate View

Communicate View is an util that associates with Communicator(). This util ir really similar to Serial Monitor in Arduino IDE that you can send and receive serial messages between your microcontroller and computer. This tool does not work with uno.js client since Communicator() supports only for your own Arduino programs that installed in your microcontroller.

Using communicate view, you should be able to type a new message and press enter to send it to the microcontroller, also change the baud rate of the communication.

You can simply create a Communicator() and connect it with CommunicateView(). Look at the following example. 

// create communicator
const comTool = new UNO.Communicator()

// create communicate view
const comView = new UNO.CommunicateView(comTool)

// create start button to start the communicator
const button = new UNO.StartButton(comTool)

// append communicate view to body
document.body.appendChild(comView.element)

// append start button to body
document.body.appendChild(button.element)

As you can see in this example, you can use the StartButton() not only for Controller(), also for the Communicator(). You can provide init and stop functions to the StartButton() constructor or it will take the default communicator start() and stop() methods. 

new UNO.StartButton(comTool, init, stop)
