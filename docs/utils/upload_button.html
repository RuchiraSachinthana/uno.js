Upload Button

Upload Button is an util that conncted with UNO.Uploader(). Using upload button, you can manage to create a fast user interface widget to upload intel HEX files according o user selection. Following upload button is created t upload the uno.js client by selecting the Arduino microcontroller board model.

Like that, you can provide a list of hex files and appropriate titles to them and represent as an upload button. Look at the following example. You have to create the Uploader() first. Then provide it as a parameter to the upload button. 

// create uploader
const uploader = new UNO.Uploader()

// create upload button
const button = new UNO.UploadButton(uploader, {
    'File #1' : { board : 'uno', file : 'path/to/file_1.hex' },
    'File #2' : { board : 'uno', file : 'path/to/file_2.hex' },
    'File #3' : { board : 'uno', file : 'path/to/file_3.hex' }
})

// append button into body
document.body.appendChild(button.element)

Above example will display the upload button with a drop down for File #1, File #2 and File #3. You can see that those are the root keys of input object structure to the upload button. Also each child object contains the board and hex file URL. These options were already explained in Uploader().
