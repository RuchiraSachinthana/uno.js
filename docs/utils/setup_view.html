Setup View

When you need to list down required components and how to connect those components with each other, you can use SetupView() to create it faster. State view even contains icons for each Arduino component that you can use anywhere in your web page.

Simply use SetupView() to create the util and provide an object as first parameter representing icon keys and label for each those items. Then as the second parameter, you have to provide a string array. This array should contain connections in each item and equal sign (=) in the middle of those pin connections. 

// create setup view
const setup = new UNO.SetupView({
    'ARD' : 'Arduino Board',
    'ACC' : 'Accelerometer'
}, [
    'ACC(GND) = GND',
    'ACC(SDA) = PIN(A4)',
    'ACC(SCL) = PIN(A5)'
])

// append setup view element to body
document.body.appendChild(setup.element)

You can use labels anyhow you need. But key values should be specific that represent icon codes from uno.js. Look at the following list of icon codes. 

ARD - Arduno Board

ACC - Accelerometer Module
LCD - Liquid Crystal Display
SRV - Servo Motor
JST - Joystick Module
BZR - Buzzer Module
LSR - Laser Module
IRR - Infrared Receiver

USS - Ultrasonic Sensor
IRS - Infrared Sensor
MIC - Microphone Sensor
BPS - Barometric Pressure Sensor
FLM - Flame Detection Sensor
SMK - Smoke Detection Sensor
PIR - Passive Infrared Sensor

LED - Light Emitting Diode
